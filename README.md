Linux Commands Used

#!/bin/bash
Specifies Bash shell interpreter.

echo
Prints text and writes poem lines into the file.

mkdir -p
Creates the poetry directory if it does not exist.

touch
Creates the poem text file.

>>
Appends text to an existing file.

cat
Displays poem contents to the terminal.

ls -l
Shows file permissions and details.

pwd
Displays current working directory.

date
Shows system date and time.

chmod +x
Makes script executable.
